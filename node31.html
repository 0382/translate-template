特征值与特征向量

多项式 p(\lambda) = {\rm det}(\lambda B-A) 被称为 特征多项式 of A - \lambda B。p(\lambda)=0 的根是 A - \lambda B 的 特征值。 由于 p(\lambda) 的次数为 n，它有 n 个根，因此 A - \lambda B 有 n 个特征值。

满足 Ax = \lambda Bx 的非零向量 x 是特征值 \lambda 的 （右）特征向量。特征对 (\lambda,x) 也满足 x^* A = \lambda x^* B，因此我们也可以称 x 为 左特征向量。

确定性矩阵束 A - \lambda B 的所有特征值都是实数。 这使得我们可以将它们按排序顺序写成 \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n。 如果所有 \lambda_i > 0，则 A - \lambda B 被称为 正定， 而如果所有 \lambda_i \geq 0，则 A - \lambda B 被称为 半正定。 负定 和 半负定 的定义 与之类似。如果既有正特征值又有负特征值， A - \lambda B 被称为 不定。

如果 A 和 B 都是实数，则每个 x_i 都是实数。 尽管 x_i 可能不唯一， 但可以选择它们全部相互 B 正交： x_i^* B x_j=0 如果 i \neq j。这也被称为 关于由 Hermitian 正定矩阵 B 诱导的内积的正交性。 当一个特征值与其他所有特征值都不同时， 其特征向量是唯一的（直至乘以标量）。
