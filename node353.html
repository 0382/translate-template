数值示例

我们为描述的示例问题提供了实现。

我们展示了这些示例及其详细输出。我们包含这些输出，以便读者可以检查其软件包副本是否正常执行。用户应在以下目录中运行 MATLAB（版本 5 或兼容版本）： >> !ls Fline.m dgrad.m grad.m move.m sg_min.m README dimension.m gradline.m nosym.m sg_newton.m clamp.m dtangent.m invdgrad_CG.m partition.m sg_prcg.m connection.m examples invdgrad_MINRES.m sg_dog.m tangent.m dFline.m finitediff ip.m sg_frcg.m （注意，可能还会有额外的子目录 docs/ 或 @cell/。）

每个示例问题在 examples 子目录中都有一个子目录。 >> !ls examples jordan ldatoy procrustes simdiag simschur tracemin 这些子目录中的每一个都包含 F(Y)（dF(Y) 和 ddF(Y,H)）的实现，它设置了 F(Y) 的全局参数（在调用任何其他函数之前必须调用此函数），一个用于生成初始猜测的 guess 函数，以及可能用于计算 F(Y)、相关量或特定问题实例的一些辅助函数。 >> !ls examples/ldatoy F.m dF.m guess.m Kinetic.m ddF.m parameters.m >> !ls examples/jordan Block.m dBlock.m ddF.m guess.m post.m F.m dF.m frank.m parameters.m

我们在支持的优化模式（'newton'、'dog'、'frcg' 和 'prcg'）和 'euclidean' 度量下执行了这些示例。对于同一问题的不同实例，不同的模式可能表现最佳，因此读者不应认为某种特定模式会在所有实例中表现出优越性能。
