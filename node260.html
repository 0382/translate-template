降阶模型的应用

在特征值计算中，通常可以自由选择带状Lanczos方法的右起始向量和左起始向量。然而，在其他重要应用中，起始向量是作为问题的一部分给出的。一个这样的应用是具有多输入和多输出的时不变线性动力系统的降阶建模；例如，参见最近的调查[176]。这类系统的特点是形式为

H(s) = C^T \left(I - s A\right)^{-1} B, \quad s\in {\mathcal C}.
 (189)

的矩阵值传递函数。这里，A 是一个一般非厄米矩阵的方阵，而
B = \left[ \begin{array}{cccc}b_1 & b_2 & \cdots & b_m\e......in{array}{cccc}c_1 & c_2 & \cdots & c_p\end{array} \right]
 (190)

是分别具有 m 和 p 列的矩形矩阵。此外，m 是输入的数量，p 是输出的数量，并且 m 和 p 通常是不同的。带状Lanczos方法（应用于 A 并以矩阵(7.84)的列，B 和 C，作为右起始向量和左起始向量）可以用来生成由(7.83)描述的线性动力系统的降阶模型。

对于降阶建模，还使用了具有负索引 k\leq 0 的条目 t_{jk} 和 \tilde{t}_{jk}。更确切地说，在这种情况下，算法7.16中的步骤(16)增加了以下六行： if j\leq m_c,
set \rho_{jk}=t_{j,k-m} for all k with j-m_c+m \leq k \leq m,
and set \rho = \left[\, \rho_{ik} \, \right]_{i=1,2,\ldots,j,\, k=1,2,\ldots,m}
if j\leq p_c,
set \eta_{jk}=\tilde{t}_{j,k-p} for all k with j-p_c+p \leq k \leq p,
and set \eta = \left[\, \eta_{ik} \, \right]_{i=1,2,\ldots,j,\, k=1,2,\ldots,p} 这里我们再次使用约定，算法7.16中未明确定义的条目 \rho_{ik} 和 \eta_{ik} 被设置为零。

矩阵 \rho 是上三角形且大小为 m_1 \times m。这里，m_1 定义为在算法7.16的迭代 j 中 m_c=m_c(j) 的值，其中 j=m_c 达到。事实证明，m_1 只是 m 减去已经被收缩的右初始向量 b_1,b_2,\ldots,b_m 的数量。特别是，m_1\leq m，并且 m_1 = m 当且仅当没有右起始向量被收缩。矩阵 \eta 是上三角形且大小为 p_1 \times p。这里，p_1 定义为在算法7.16的迭代 j 中 p_c=p_c(j) 的值，其中 j=p_c 达到。数 p_1 是 p 减去已经被收缩的左初始向量 c_1,c_2,\ldots,c_p 的数量。特别是，p_1\leq p，并且 p_1 = p 当且仅当没有左起始向量被收缩。\rho 的条目是用来将右起始向量转换为前 m_1 个右Lanczos向量的系数，\eta 的条目是用来将左起始向量转换为前 p_1 个左Lanczos向量的系数。

现在设 j\geq \max\{m_1,p_1\}，并设 T_j^{\rm (pr)}，\rho，和 \eta 是在算法7.16的 j 次迭代后生成的矩阵。这三个矩阵然后定义了原始传递函数(7.83)的 j 阶降阶模型，如下所示：

H_j(s) = \eta_j^T D_j \left(I_j - s T_j^{\rm (pr)}\right)^{-1} \rho_j,\quad s\in {\mathcal C}.
 (191)

这里，

\rho_j = \left[ \begin{array}{c}\rho\\0\end{array} \......{c}\eta\\0\end{array} \right] \in {\mathcal C}^{j\times p}

是矩阵 \rho 和 \eta，分别添加了 j-m_1 和 j-p_1 行零。降阶模型(7.85)可以被证明是原始传递函数(7.83)的某种矩阵-Padé逼近；参见[176]及其中的参考文献。
