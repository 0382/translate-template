特征分解

我们将讨论四种特征分解，或者说四种与矩阵 A 相似且特征问题求解更为简单的矩阵。第一种是对角形式，仅在存在 n 个独立特征向量时存在。第二种是约旦形式，它将对角形式推广到所有矩阵。它可能非常病态（实际上，它可能会不连续变化），因此在数值计算中我们通常使用第三种，即舒尔形式，它在计算上更经济且更稳定。我们简要提及第四种，称之为约旦-舒尔形式，它与舒尔形式一样稳定，但能计算出约旦形式所提供的关于不变子空间的某些详细信息。

定义 \Lambda = {\rm diag}(\lambda_1 ,\ldots, \lambda_n )。 如果存在 n 个独立的右特征向量 x_1,\ldots,x_n， 我们定义 X = [x_1,\ldots,x_n]。 X 被称为 A 的特征向量矩阵。 n 个等式 A x_i = \lambda_i x_i 对于 i=1,\ldots,n 也可以写成 AX = X \Lambda 或 A = X \Lambda X^{-1}。这种分解

A = X \Lambda X^{-1}

被称为 A 的对角形式 。注意 X^{-1}A = \Lambda X^{-1}。 令 y_i^* 表示 X^{-1} 的第 i 行， 我们可以看到 y_i^*A = \lambda y_i^*。换句话说， X^{-1} 的行是 A 的左特征向量。 因此，如果 A 有 n 个独立的特征向量，那么它与对角矩阵 \Lambda 相似。在这种情况下，我们称 A 是可对角化的。

如果我们取 X 的 k 列（例如 \hat{X}= X(:,[2,3,5]) = 第 2、3 和 5 列），这些列张成 A 的一个不变子空间。 如果我们取相应的子矩阵 \hat{\Lambda}= {\rm diag}(\lambda_2 , \lambda_3 , \lambda_5 ) 和 X^{-1} 的相应三行（例如 \hat{Y}^* = X^{-1}([2,3,5],:)）， 我们可以写出相应的部分对角形式为 A \hat{X}= \hat{X}\hat{\Lambda} 或 \hat{Y}^* A \hat{X}= \hat{\Lambda}。如果 \hat{X} 中的列被张成相同不变子空间的 k 个不同向量替换，我们会得到一个不同的部分特征分解 A \check{X}= \check{X}\check{A}， 其中 \check{A} 是一个 k 阶矩阵，其特征值与 \hat{\Lambda} 相同，尽管 \check{A} 可能不是对角矩阵。对于下面讨论的其他特征分解，也可以采用类似的步骤来生成部分特征分解。

如果所有的 \lambda_i 都是不同的，那么存在 n 个独立的特征向量， 并且对角形式存在。这是最简单和最常见的情况。 例如，如果随机选择一个矩阵，特征值不同的概率是 1。

矩阵 A(0) 在 (2.3) 中的对角形式不存在， 因为它只有一个独立的特征向量。相反，我们可以计算它的约旦形式，这是一种分解

A = XJX^{-1},

其中 J 是一个块对角矩阵，即形式为 J = {\rm diag} (J_1 ,\ldots,J_e)，其中每个 J_i 是一个被称为约旦块的方阵。 每个 J_i 是上三角矩阵，其单个特征值 \lambda_i 在主对角线上，且第一上对角线为 1。 J_i 有一个独立的右特征向量和一个独立的左特征向量。 A(0) 是一个 2 阶约旦块，其单个特征值为 0， 而 A(0) + \lambda_i I 是一个 2 阶约旦块，其单个特征值为 \lambda_i。 可以证明，约旦形式明确描述了 A 的所有可能不变子空间，这些子空间由 X 的某些列子集张成 [187]。

不幸的是，约旦形式通常不适用于数值计算。这里有两个原因。首先，它可能随着 A 的变化而连续变化。例如，矩阵 J 对于 A(\epsilon) 在 \epsilon \neq 0 时是 {\rm diag}(+ \epsilon , - \epsilon)，但对于 A(0)，J=A(0) 本身。其次，特征向量矩阵 X 可能非常病态，即难以准确求逆。 在 A(\epsilon) 的情况下，X 的条件数会像 1/\epsilon 那样增长。

因此，我们使用形式为

A = QTQ^*,

的特征分解，其中 Q 是一个酉（或正交）相似变换，T 是三角（或拟三角）矩阵。这被称为 A 的舒尔形式。 当 A 是复数，或实数且所有特征值都是实数时， T 是三角矩阵，其对角线上是 A 的特征值。 当 A 是实数且有复数特征值，而 Q 是实数且正交时， T 不能是实数和三角矩阵。相反，我们允许 T 的对角线上出现具有复数特征值的 2 阶块，我们称之为拟三角形式。 为了简化表述，我们将考虑复数情况，因此 T 是三角矩阵。令 \lambda_1 = T_{11},\ldots,\lambda_n = T_{nn} 为 T 对角线上的特征值顺序；每种顺序都有一个不同的舒尔形式。 Q 的列被称为舒尔向量。舒尔形式不像对角形式和约旦形式那样明确给出特征向量和所有不变子空间的基础，但它可以相当稳定地计算。 Q 的前 k 列张成特征值 \lambda_1 到 \lambda_k 的不变子空间，但所有其他特征空间需要计算。例如，特征向量可以通过从 T 中解一个三角方程组并乘以 Q 来计算 [114]。

最后，我们考虑约旦-舒尔形式。 它相当复杂，因此我们在这里只总结其特性。与舒尔形式一样，它只使用酉（正交）变换，因此可以稳定地计算。 与约旦形式一样，它明确显示了约旦块的大小，并给出了比舒尔形式更多的不变子空间的明确基础。

许多教科书给出了诸如在 A 的约旦形式基础上计算指数矩阵 e^A 等问题的显式解 [114]。 这些方法在数值上应避免使用，因为计算约旦形式很困难。 几乎所有这些问题都有基于舒尔形式或某些情况下的约旦-舒尔形式的替代解决方案。
