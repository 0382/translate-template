特征分解

我们讨论四种特征分解，或者说四种等价于A - \lambda B的矩阵束，这些分解使得求解特征问题更为简单。本节类似于第2.5.4节。

第一种特征分解，对角形式，仅在存在n个独立特征向量时存在。第二种，魏尔斯特拉斯形式，将对角形式推广到所有特征多项式p(\lambda)不恒为零的矩阵束。我们也可以将魏尔斯特拉斯形式描述为将约旦形式矩阵推广到矩阵束。魏尔斯特拉斯形式，如同约旦形式，可能非常病态（实际上，它可能不连续变化），因此在数值计算中我们通常使用第三种，广义舒尔形式，它在计算上更经济且更稳定。我们简要提及第四种，我们称之为魏尔斯特拉斯-舒尔形式，它与舒尔形式一样稳定，但计算了一些由魏尔斯特拉斯形式提供的关于压缩子空间的详细信息。

定义\Lambda = {\rm diag}(\lambda_1 ,\ldots, \lambda_n )。如果存在n个独立的右特征向量x_1,\ldots,x_n，我们定义X = [x_1,\ldots,x_n]。X被称为A的（右）特征向量矩阵。类似地，令Y = [y_1,\ldots,y_n]为左特征向量矩阵。2n个等式A x_i = \lambda_i B x_i和y_i^* A = \lambda_i y_i^* B对于i=1,\ldots,n也可以写成 AX = BX \Lambda和 Y^*A = \Lambda Y^* B。 X和Y还可以选择使得 Y^*AX = \Lambda_A和 Y^*BX = \Lambda_B均为对角矩阵，并且 \Lambda = \Lambda_A \Lambda_B^{-1}。分解

A - \lambda B = Y^{-*} ( \Lambda_A - \lambda \Lambda_B) X^{-1}

被称为A - \lambda B的对角形式。 在这种情况下，我们称A - \lambda B为可对角化。

如果我们取X和Y的k列（例如\hat{X}= X(:,[2,3,5]) = 第2、3、5列和\hat{Y}= Y(:,[2,3,5]))，那么这些列张成了A - \lambda B的一对压缩子空间。如果我们取相应的子矩阵 \hat{\Lambda}_A = {\rm diag}(\lambda_{A,2} , \lambda_{A,3} , \lambda_{A,5} )和 \hat{\Lambda}_B = {\rm diag}(\lambda_{B,2} , \lambda_{B,3} , \lambda_{B,5} )， 那么我们可以写出相应的部分对角形式为 \hat{Y}^* A \hat{X}= \hat{\Lambda}_A和\hat{Y}^* B \hat{X}= \hat{\Lambda}_B。 如果\hat{X}和\hat{Y}的列被替换为张成相同压缩子空间的k个不同向量，那么我们得到一个不同的部分特征分解 \check{Y}^* A \check{X}= \check{A}和\check{Y}^* B \check{X}= \check{B}， 其中\check{A}- \lambda \check{B}是一个k乘k的矩阵束，其特征值是 \hat{\Lambda}= \Lambda_A \Lambda_B^{-1}的特征值，尽管 \check{A}- \lambda \check{B}可能不是对角矩阵。对于下面讨论的其他特征分解，类似的产生部分特征分解的方法也适用。

如果所有的\lambda_i都是不同的，那么存在n个独立的特征向量，并且对角形式存在。这是最简单和最常见的情况。例如，如果随机选择A和B，特征值不同的概率为1。

在第2.5.4节中，矩阵束A(0) - \lambda I的对角形式不存在，因为它只有一个独立的特征向量。相反，我们可以计算它的魏尔斯特拉斯形式，这是一个分解

A - \lambda B = Y^{-*}(J_A - \lambda J_B)X^{-1},

其中J_A - \lambda J_B是一个块对角矩阵束，每个特征值有一个或多个上三角对角块。当没有无穷特征值时，这与第2.5.4节中讨论的约旦形式相同。当存在无穷特征值时，存在一些类似于约旦块的块，具有单个无穷特征值和一个右和一个左特征向量。可以证明，魏尔斯特拉斯形式明确描述了A - \lambda B的所有可能的压缩子空间，这些子空间由X和Y的某些列的子集张成[187]。

不幸的是，魏尔斯特拉斯形式通常不适合数值计算，原因与约旦形式不适合相同。参见第2.5.4节的讨论。

因此，我们使用形式的特征分解

A - \lambda B = Z(T_A - \lambda T_B)Q^*,

其中Q和Z是酉（或正交）矩阵，T_A - \lambda T_B是三角（或拟三角）矩阵。这被称为A - \lambda B的广义舒尔形式 。 当A - \lambda B是复数，或实数且所有特征值都是实数或无穷大时，T_A - \lambda T_B是三角矩阵，其对角线上是A - \lambda B的特征值。当A - \lambda B是实数且具有复数特征值，且Q是实数和正交矩阵时，T_A - \lambda T_B不能是实数和三角矩阵。相反，我们允许在对角线上出现具有复数特征值的2乘2块，我们称之为拟三角形式。为了简化表述，我们将考虑复数情况，因此T_A - \lambda T_B是三角矩阵。令\lambda_1 = T_{A,11}/T_{B,11} ,\ldots,\lambda_n = T_{A,nn}/T_{B,nn}为对角线上出现的特征值顺序；每种顺序都有一个不同的广义舒尔形式。Q和Z的列被称为广义舒尔向量。广义舒尔形式不像对角形式和魏尔斯特拉斯形式那样明确给出特征向量和所有压缩子空间的基，但它可以非常稳定地计算。Q和Z的前k列张成了特征值\lambda_1到\lambda_k的右和左压缩子空间，但所有其他特征空间需要计算。例如，右特征向量可以通过从T_A - \lambda T_B解一个三角方程组并乘以Q来计算。

最后，我们考虑魏尔斯特拉斯-舒尔形式。 它相当复杂，因此我们在这里只总结其性质。与舒尔形式一样，它仅使用酉（正交）变换，因此可以稳定计算。与魏尔斯特拉斯形式一样，它明确显示了（约旦）块的大小，并给出了比舒尔形式更多的不变子空间的明确基。

许多教科书给出了诸如求解常微分方程 B \dot{x} (t) = A x(t)在A - \lambda B的魏尔斯特拉斯形式下的显式解[114]。这些方法在数值上应避免使用，因为计算魏尔斯特拉斯形式的困难。几乎所有这些问题都有基于广义舒尔形式或某些情况下的魏尔斯特拉斯-舒尔形式的替代解。
