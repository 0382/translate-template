相关特征问题

由于HEP是理解最为透彻的特征问题之一，因此识别其他特征问题何时可以转换为HEP是非常有帮助的。

  1. 如果A是非厄米矩阵，但\hat{A}= \alpha SAS^{-1}对于容易确定的\alpha和S是厄米矩阵，那么计算\hat{A}的特征值\hat{\lambda}和特征向量\hat{x}可能是明智的。通过以下转换可以将这些特征值和特征向量转换为A的特征值\lambda和特征向量x： \lambda = \hat{\lambda}/ \alpha 和 x = S^{-1} \hat{x}。 例如，将一个斜厄米矩阵A（即，A^* = -A）乘以常数\sqrt{-1}会使其变为厄米矩阵。进一步讨论见§2.5。

  2. 如果A = B^*B对于某个矩形矩阵B，那么A的特征问题等价于B的奇异值分解（SVD），讨论见§2.4。假设B是m乘n的，因此A是n乘n的。一般来说，如果A的大小与B相当或更小（即n \leq m，或稍大一些），那么A的特征问题通常比B的SVD更经济。但计算A的小特征值可能不如计算B的小奇异值准确。进一步讨论见§2.4。

  3. 如果有一个广义HEP Ax = \lambda Bx，其中A和B是厄米矩阵且B是正定的，那么它可以转换为厄米特征问题，如下所示。首先，分解B = LL^*，其中L是任意非奇异矩阵（这通常使用Cholesky分解完成）。然后求解\hat{A}= L^{-1}A L^{-\ast}的HEP。\hat{A}和A - \lambda B的特征值相同，并且如果\hat{x}是\hat{A}的特征向量，那么x = L^{-\ast}\hat{x}满足Ax = \lambda Bx。进一步讨论见§2.3。
