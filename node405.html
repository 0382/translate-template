带凯莱变换的雅可比-戴维森方法

雅可比-戴维森方法在第§7.12节中进行了讨论。 雅可比-戴维森方法与戴维森方法的不同之处在于，需要求解的线性系统被投影到与当前Ritz向量正交的空间上。 这导致了修正方程的求解

\left( I - \frac{B x_j x_j^{\ast}}{x_j^{\ast} B x_j} \right)......_j^{\ast}}{x_j^{\ast} x_j} \right) w_j = (A - \theta_jB) x_j,
 (276)

其中 (\theta_j,x_j) 是第 j 次迭代的 Ritz 对。 （注意通常在右侧残差前会有一个负号。） 假设 w_j 与 x_j 正交，即 ( I - x_j x_j^{\ast} / x_j^{\ast} x_j) w_j = w_j。 当使用不精确求解器时，我们有一个满足以下条件的残差 s_j

s_j = (A - \theta_j B) x_j- \left( I - \frac{B x_j x_j^{\ast} }{x_j^{\ast} B x_j}\right)(A - \theta_j B) w_j \ .

注意，由于假设 w_j 与 x_j 正交，投影在 w_j 前的部分被省略了。 我们可以将其改写为




其中 \epsilon_j = -(x_j^{\ast} (A -\theta_j B) w_j) / ( x_j^{\ast} B x_j)。

换句话说，修正方程的解 w_j 是通过凯莱变换 T_{\mathrm{C}} = (A-\theta_j B)^{-1} (A-(\theta_j+\epsilon_j)B) 作用于最近的 Ritz 向量得到的。 示例 11.2.1 在 [411] 中表明，随着收敛，\epsilon_j 趋向于零。 凯莱变换的极点和零点都接近所需的特征值。 这满足了 Ax = \lambda Bx 和 T_{\mathrm{IC}} u = \eta u 的特征向量之间良好匹配的条件， 这在第§11.2.2节的末尾有所阐述。

以下观察有点有趣。 由于 w_j = -\epsilon_j (A-\theta_j B)^{-1} B x_j + x_j， 当 s_j=0 时，我们从 x_j^{\ast} w_j=0 得到

\theta_j + \epsilon_j = \theta_j+ \frac{x_j^{\ast} x_j}{x_j^{\ast} (A-\theta_j B)^{-1} B x_j} \ .

因此，凯莱变换的极点是 x_j 的瑞利-里兹商，零点是目标为 \theta_j 的调和瑞利-里兹商。
