相关特征问题

  1. 如果 A 和 B 是厄米特矩阵，B 不是正定矩阵，但对于某些实数 \alpha 和 \beta，\alpha A + \beta B 是正定矩阵，则可以求解广义厄米特特征问题 A - \lambda (\alpha A + \beta B)。设 \hat{B}= \alpha A + \beta B，则 A - \lambda B 和 A - \lambda \hat{B} 的特征向量相同。A - \lambda B 的特征值 \lambda_i 和 A - \lambda \hat{B} 的特征值 \hat{\lambda}_i 之间的关系为 \lambda_i = \beta \hat{\lambda}_i / (1 - \alpha \hat{\lambda}_i)。

  2. 如果 A 和 B 不是厄米特矩阵，但 \hat{A}= \alpha SAT 和 \hat{B}= \beta SBT 是厄米特矩阵，且 \hat{B} 是正定矩阵，对于容易确定的 \alpha、\beta 以及非奇异矩阵 S 和 T，则可以计算 \hat{A}- \lambda \hat{B} 的特征值 \hat{\lambda} 和特征向量 \hat{x}。通过 \lambda = \hat{\lambda}\beta / \alpha 和 x = T \hat{x} 可以将这些转换为 A 的特征值 \lambda 和特征向量 x。例如，如果 B 是厄米特正定矩阵，但 A 是斜厄米特矩阵（即 A^* = -A），则 \sqrt{-1}A 是厄米特矩阵，因此可以选择 \alpha = \sqrt{-1}、\beta=1 和 S=T=I。进一步讨论见第§2.5节。

  3. 如果有一个广义厄米特特征问题 Ax = \lambda Bx，其中 A 和 B 是厄米特矩阵且 B 是正定矩阵，则可以将其转换为厄米特特征问题。首先，分解 B = LL^*，其中 L 是任意非奇异矩阵（通常使用 Cholesky 分解）。然后求解 \hat{A}= L^{-1}A L^{-\ast} 的厄米特特征问题。\hat{A} 和 A - \lambda B 的特征值相同，如果 \hat{x} 是 \hat{A} 的特征向量，则 x = L^{-\ast}\hat{x} 满足 Ax = \lambda Bx。实际上，这是一个标准的 A - \lambda B 算法。

  4. 如果 A 和 B 是正定矩阵，且 A = R^*R 和 B = U^*U 对于某些矩形矩阵 R 和 U，则 A - \lambda B 的特征问题等价于 R 和 U 的商奇异值分解（QSVD），讨论见第§2.4节。算法的状态表明，尝试求解 A - \lambda B 的特征问题可能比计算 R 和 U 的 QSVD 更好。
