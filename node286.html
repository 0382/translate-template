移位与反转

将问题转换为标准形式的所有变换，如（8.3）、（8.4）或（8.5），在A和B都奇异或B条件数不佳时均无法使用。一种吸引人的流行技术是首先对原始问题进行移位，然后进行分裂反转。这就是SI方法，如第3.3节所述。具体来说，设\sigma为用户选定的移位值，使得矩阵A-\sigma B非奇异；那么原始问题（8.1）可以转换为
C x = \mu x\quad \mathrm{and} \quadz^{\ast} C = \mu z^{\ast},
 (219)

其中

C = (A - \sigma B)^{-1} B, \quad\mu = \frac{1}{\lambda - \sigma}, \quad \mathrm{and} \quadz= (A - \sigma B)^{\ast} y.

我们看到，最接近移位\sigma的问题（8.1）的特征值\lambda被映射为简化标准特征值问题（8.6）的外部特征值，即最大幅值的特征值，这些特征值首先被迭代方法很好地近似。

在实践中，有效的移位选择取决于用户的偏好和对底层广义特征问题的了解。一个好的移位不仅放大了所需的特征值，还导致了一个条件良好的矩阵A-\sigma B。这往往使得选择好的移位成为一个挑战。

对于简化标准特征值问题（8.6）的迭代方法应用，需要计算矩阵向量乘积

r = C q = \left( (A - \sigma B)^{-1} B \right) q

或

s = C^{\ast} p = \left( (A - \sigma B)^{-1} B \right)^{\ast} p

对于给定的向量q和p。为了高效计算，设
A - \sigma B = {\mathcal L} {\mathcal U}
 (220)

表示A-\sigma B的某种方便的分解，其中{\mathcal L}和{\mathcal U}是方阵。由于假设A-\sigma B是非奇异的，因此因子{\mathcal L}和{\mathcal U}也是非奇异的。分解应选择使得与{\mathcal L}、{\mathcal L}^{\ast}和/或{\mathcal U}、{\mathcal U}^{\ast}相关的线性方程组可以高效求解，通常使用稀疏LU分解。参见第10.3节。当然，如果这导致方便的线性系统，也可以选择{\mathcal L} = A - \sigma B和{\mathcal U} = I。

利用上述分解，矩阵向量乘积r = Cq可以按以下步骤计算： (a) 形成v = B q，
(b) 解{\mathcal L} w = v求w，
(c) 解{\mathcal U} r = w求r。 类似地，矩阵向量乘积s = C^{\ast} p可以按以下三个步骤计算： (a)^{\prime} 解{\mathcal U}^{\ast} v = p求v，
(b)^{\prime} 解{\mathcal L}^{\ast} w = v求w，
(c)^{\prime} 形成s = B^{\ast} w。

SI技术是处理广义特征问题（8.1）的强大工具。主要问题，往往成为瓶颈，是找到A-\sigma B的方便分解（8.7），使得相关的线性方程组可以高效求解。如果用A-\sigma B求解线性方程组的精度变得过于昂贵，那么可以考虑使用不精确的Cayley变换（参见第11.2节）或Jacobi-Davidson方法。
