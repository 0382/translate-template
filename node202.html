相关问题
  J. Demmel

如第§2.4.7节所述，存在一些与SVD紧密相关的问题，其解决方案将在本节中描述。这份清单并不详尽，因为使用低秩矩阵近似（其中“最优”近似由截断SVD提供）在应用中非常普遍。根据具体应用，可能适合使用此处描述的SVD算法或其他低秩近似方法。

  1. **商或广义SVD** 对于矩阵A和B，定义如下。首先假设A是m行n列，B是n行n列且非奇异。设AB^{-1}的SVD为U \Sigma V^*。那么我们也可以将A和B分解为A = U \Sigma_A X和B = V \Sigma_B X，其中X是n行n列且非奇异， \Sigma_A = {\rm diag} (\sigma_{A,1},\ldots,\sigma_{A,n})是m行n列， \Sigma_B = {\rm diag} (\sigma_{B,1},\ldots,\sigma_{B,n})是n行n列， 且\sigma_{A,i}^2 + \sigma_{B,i}^2 = 1。 那么\Sigma = \Sigma_A \Sigma_B^{-1} = {\rm diag} (\sigma_1,\ldots,\sigma_n)。 值\sigma_i = \sigma_{A,i} / \sigma_{B,i}被称为A和B的**广义奇异值**。

    使用变换方法（适用于稠密矩阵）的软件可在LAPACK驱动程序xGGSVD中找到，适用于更一般的情况，即B为奇异或p行n列。在MATLAB中，也可通过命令gsvd获得。ScaLAPACK尚无相关软件。使用这些算法的优势仅在于精度；即当B条件数较差时，它们比直接计算AB^{-1}的SVD要精确得多。但它们也相对较慢，因此如果B条件良好，直接计算AB^{-1}的SVD会更好。

    当A和B为大型稀疏矩阵且B为方阵、非奇异且可分解时，建议应用上述仅需要乘以AB^{-1}及其转置的SVD算法。

    如果B不是方阵，建议使用第5章中的技术，求解广义Hermitian特征值问题A^*A - \lambda B^*B的特征值和特征向量。这是因为A^*A - \lambda B^*B的特征分解为 X^*A^*AX = \Sigma_A^* \Sigma_A和 X^*B^*BX = \Sigma_B^* \Sigma_B。

  2. 还可以为AB（而非AB^{-1}）定义**更多广义SVD**（即**乘积SVD**），或者形式为A_1^{\pm 1} A_2^{\pm 1} \cdots A_k^{\pm 1}的任意乘积。LAPACK、ScaLAPACK或MATLAB中没有针对这些情况的专用软件。因此，在稠密情况下，需要显式形成这些乘积（尽管已有算法提出避免这种情况[54,3,53]），而在稀疏情况下，则无需形成它们即可进行乘法运算。

  3. 寻找使\Vert Ax - b \Vert _2最小化的x，即**线性最小二乘问题**。 假设A是m行n列且m>n；那么我们称最小二乘问题为**超定**。如果A是非奇异的， 且A = U_n \Sigma_n V_n^*是A的瘦SVD，那么解为x = V_n \Sigma_n^{-1} U_n^*b。通常，解法方程A^*Ax = A^*b或使用QR分解A = QR计算x = R^{-1}Q^*b更经济，但当A条件较差，即\sigma_n \ll \sigma_1时，SVD可以更精确。特别是，当A非常病态时，通常使用A的截断SVD而非瘦SVD：x = V_t \Sigma_t^{-1} U_t^*b。

    解决最小二乘问题时，无需显式计算A的瘦SVD或截断SVD。对于稠密问题，LAPACK驱动程序xGELSD是首选方法，使用分治法快速求解最小二乘问题，无需显式形成瘦SVD。在ScaLAPACK中，只有计算瘦SVD的程序PxGESVD，可用于最小二乘问题。对于稀疏问题，可以使用从方程(6.7)或(6.8)导出的近似分解A \approx U_k B_k V_k^*来求解x = V_k B_k^{-1} U_k^{-1}。
