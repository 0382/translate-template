按照算法7.8构造的正交矩阵 Q 具有非常特殊的形式，可以写成 Q = R + y e_1^*, with R e_1 = 0 , R^* y = 0, 其中 R 是上三角矩阵。它也可以写成 Q = L + y g^* , with L e_1 = 0 , L^* y = e_1 - g, 其中 L 是下三角矩阵， 并且 g^* \equiv e_1^* + \frac{1}{\eta_1} e_1^* R 。

现在，考虑矩阵 Q^* H Q。从 Q^* = (L + y g^* )^*, Q = (R + y e_1^* ) 来自（7.19）和（7.18）将得到 Q^* H Q &=& Q^* H ( R + y e_1^*)
&=& (L^* + g y^*) H R + Q^*Hy e_1^*
&=& L^*HR + g y^*H R + Q^*Hy e_1^*. 由于 L^* 和 R 都是上三角矩阵，因此 L^*HR 是上海森堡矩阵，其第一行和第一列均为零 由于 Le_1 = Re_1 = 0。
