线性形式的转换

不难看出，QEP（二次特征值问题）在9.1中与以下广义的“线性”特征值问题是等价的：
A z = \lambda B z \quad \mathrm{and} \quad w^{\ast} A = \lambda w^{\ast} B ,
 (248)

其中
A = \twobytwo{0}{I}{-K}{-C}, \quad B = \twobytwo{I}{0}{0}{M}
 (249)

以及
z = \twobyone{x}{\lambda x}, \quad w = \twobyone{(\lambda M + C)^{\ast} y }{y}.
 (250)

广义特征值问题9.4通常被称为QEP9.1的线性化。可以证明，对于任何上述形式的矩阵A和B，右特征向量z和左特征向量w具有9.6中所描述的结构。

注意，从因式分解

A - \lambda B = \twobytwo{0}{I}{-I}{-\lambda M - C}\twobytwo{\lambda^2 M + \lambda C + K}{0}{0}{I}\twobytwo{I}{0}{-\lambda I }{I},
 (251)

我们可以得出结论，矩阵束A - \lambda B与矩阵是等价的，即
\twobytwo{\lambda^2 M + \lambda C + K }{0}{0}{I}
 (252)

并且

{\rm det}(A - \lambda B ) = {\rm det}(\lambda^2 M + \lambda C + K).

这意味着原始QEP9.1的特征值与广义特征值问题9.4的特征值一致。此外，我们有：
  • L(\lambda)是正则的当且仅当A - \lambda B是正则的；
  • 如果M（从而B）是非奇异的，则L(\lambda)是正则的；
  • 如果K（从而A）是非奇异的，则L(\lambda)是正则的。
关于正则矩阵束(A,B)的理论，参见例如[425, Chap. VI]。我们将在本节中假设至少M是非奇异的。

上述转换为线性形式的一个缺点是，如果矩阵M、C和K都是埃尔米特矩阵，那么在简化形式9.5中，A并不是埃尔米特矩阵。这可以通过以下方式修复。

实际上，9.4中的矩阵对(A,B)可以选择更一般的形式

A = \twobytwo{0}{W}{- K}{ - C}, \quad B = \twobytwo{W}{0}{0}{ M},

其中W可以是任意非奇异矩阵。注意，现在矩阵束A - \lambda B与矩阵多项式9.8等价当且仅当W是非奇异的，并且由于9.7，

\det( A - \lambda B ) = \det(W)\cdot \det( \lambda^2 M + \lambda C + K ).

例如，如果矩阵M、K和C都是对称的，如特殊情况9.2所示，且K是非奇异的，那么我们可以选择W = - K，这将导致以下对称广义“线性”特征值问题
A z = \lambda B z \quad \mathrm{and} \quad w^{\ast} A = \lambda w^{\ast} B ,
 (253)

其中
A = \twobytwo{0}{-K}{-K}{ -C}, \quad B = \twobytwo{-K}{0}{0}{M}, \quad
 (254)

以及
z = \twobyone{x}{\lambda x}, \quad w = \twobyone{y}{\bar\lambda y}.
 (255)

A和B都是对称的，但可能是非定的。
