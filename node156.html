引言

广义厄米特特征值问题（GHEP）由以下方程给出：
ax=\lambda B x\,,
 (66)

其中 A 和 B 是厄米特矩阵，即 A^{\ast} = A，B^{\ast} = B。 我们称 (5.1) 中的矩阵对 \{A,B\} 为矩阵束。 在本章中，我们进一步假设 A 或 B 或 对于某些标量 \alpha 和 \beta，\alpha A + \beta B 是正定矩阵，在这种情况下，我们讨论厄米特正定束。 这一假设在许多实际重要情况下成立，且其理论与第4章阐述的标准厄米特特征值问题密切相关。 如果不存在正定组合，我们可以将 \{A,B\} 视为一般束，并使用第8章描述的理论和算法。

非标准情况，即 \alpha A + \beta B 为正定矩阵，可以通过注意到当 B 为正定时，束

(A-\theta(\alpha A + \beta B))x=0
 (67)

具有特征值 \theta_i=\lambda_i/(\beta+\alpha \lambda_i) 和与原束 (5.1) 相同的特征向量。 可以对这一修改后的束应用适用于正定 B 的任何算法，并从 \theta_i 恢复 \lambda_i。

GHEP (5.1) 具有 n 个实特征值 \lambda_i，我们可以按递增顺序排列它们，使得 \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n。与标准情况一样，多个特征值可能重合，但某些特征值可能是无穷大。如果矩阵 A 和 B 是正定矩阵，则 \lambda_1>0，但如果 A 是半正定矩阵，我们只能说 \lambda_1\geq 0。

当束 \{A,B\} 是厄米特正定时，可以找到 n 个相互 B-正交的特征向量 x_i,\,i=1,\dots,n，使得

A=X \Lambda X^{\ast},\quad B=X X^{\ast},\pagebreak
 (68)

其中 \lambda=\diag(\lambda_i) 且 X=[x_1,x_2,\dots,x_n]。特征向量 x_i 不是唯一的，但每个不同的特征值都有一个唯一的约化子空间。对于厄米特正定束，约化子空间的维数与特征值的重数相同。重要的是要记住，当一对特征值重合时，它们的特征向量失去个性：无法说一组向量是多重特征值的特征向量。

有些情况下矩阵 B 是奇异的（半正定）。只要 Ax\neq 0 对于 B 的零空间中的所有向量 x 成立，我们仍然有一个厄米特正定束。此时，束 (5.1) 具有 p 重无穷大特征值，其约化子空间为 B 的零空间。在大多数实际情况下，我们不关心计算这些无穷大特征值，因为它们对应于由特征值计算引起的物理应用中的约束、对称性或刚体模式。

A - \lambda B 的特征值 \{\lambda_i\} 可能是良态的或病态的。如果 B=I（或接近它），特征值是良态的（接近它），如同厄米特特征值问题。但如果 \vert x^{\ast}_i B x_i\vert 很小，其中 x_i 是 \lambda_i 的单位特征向量，\lambda_i 可能非常病态。我们在第5.7节中更详细地介绍了计算特征值和特征向量的误差评估。
