常规问题与奇异问题

让我们从考虑 2 \times 2 广义特征值问题 Ax = \lambda Bx 开始，其中

A = \left[ \begin{array}{cc}\alpha_1 & 0 \\0 & \alpha_2 \\\end{array} \right], \quad B = \left[ \begin{array}{cc}\beta_1 & 0 \\0 & \beta_2 \\\end{array} \right].

A - \lambda B 的特征值是 (\alpha_1, \beta_1) 和 (\alpha_2, \beta_2) 这对值，相应的特征向量分别是 x_1 = [1 \quad 0]^T 和 x_2 = [0 \quad 1]^T。如果 \beta_i 不为零，则 \lambda_i = \alpha_i/\beta_i 是一个有限特征值。否则，如果 \beta_i 为零，则 \lambda_i = \infty 是矩阵对 \{A,B\} 的一个特征值。但如果例如 \alpha_2 = \beta_2 = 0 呢？那么 {\rm det}(A - \lambda B) 对所有 \lambda 都为零，这意味着我们有一个奇异特征值问题。在这种情况下，我们有 Ax_2 = Bx_2 = 0，即 A 和 B 有一个共同的零空间。我们称 x_2 是一个不确定特征值 0/0 的特征向量。注意，共同的零空间是具有奇异特征值问题的充分条件，但不是必要条件。

最常见的广义特征值问题 Ax = \lambda Bx 是 常规的；即，A 和 B 是方阵，并且特征多项式 p_m(\lambda) = {\rm det}(A - \lambda B) 仅在有限个值处消失，其中 m 表示多项式的次数。相应的 A - \lambda B 被称为常规矩阵束。常规束的 n 个特征值是扩展复平面 {\mathcal C} \cup \infty 上的点。特征值 \lambda_i 定义为 p_m(\lambda) 的零点，以及 n - m 个额外的 \infty 特征值。

矩阵束的另一种表示形式是 叉积形式：即矩阵集 \beta A-\alpha B，其中 (\alpha,\beta) \in {\mathcal C}^2。映射 (\alpha,\beta) \mapsto \alpha/\beta 显示了 \beta A-\alpha B 和 A - \lambda B 的特征值之间的关系。例如，零和无限特征值分别表示为 (0, \beta) 和 (\alpha, 0)，并且可以像 {\mathcal C}^2 中的其他点一样处理。

如果 {\rm det}(A - \lambda B)（以及 {\rm det}(\beta A - \alpha B)）对所有 \lambda（以及对 (\alpha, \beta) 对）恒为零，那么 A - \lambda B 被称为 奇异的，并且 \{A,B\} 是一个奇异矩阵对。奇异性 \{A,B\} 的标志是某些 \alpha = \beta = 0。在存在舍入误差的情况下，\alpha 和 \beta 可能非常小。在这些情况下，特征值问题是病态的，并且其他一些计算出的非零 \alpha 和 \beta 值可能是不确定的。这类问题在第 §8.7.4 节中进一步讨论并通过示例说明。此外，矩形矩阵对是奇异的，相应的 A - \lambda B 是一个奇异束。
