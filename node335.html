逆QEP问题

对于求解广义特征值问题的大多数迭代方法， 公式 (9.4)，无论是与(9.5)还是与 (9.10)结合，都适合于确定少数 外部特征值和特征向量。如果需要计算 一些模最小的特征值和特征向量，那么 显然的变换是 \mu=1/\lambda，并且在将 QEP (9.1) 乘以 \mu^2 后，我们得到 逆QEP：
\left( M + \mu C + \mu^2 K \right) x = 0 .
 (256)

这里假设 0 不是原QEP (9.1) 的特征值，即 K 是非奇异的。

三元组 \{K,C,M\} 的QEP可以线性化，如 在§9.2.2 中讨论的那样， 例如，如 (9.4) 与 (9.5) 所示，其中 M 与 K 互换。我们可以用 \lambda 而不是 \mu 来重新表述这个广义线性化特征值问题， 这导致

A z = \frac{1}{\lambda} B z,
 (257)

其中
A = \twobytwo{-C}{-M}{I}{0}, \quadB = \twobytwo{K}{0}{0}{I}, \quadz = \twobyone{x}{\lambda x}.
 (258)

注意，从因式分解

B - \lambda A =\twobytwo{I}{\lambda M}{0}{I}\twobytwo{\lambda^2 M + \lambda C + K}{0}{0}{I}\twobytwo{I}{0}{-\lambda I}{I}

我们知道矩阵束 B - \lambda A 等价于

\twobytwo{\lambda^2 M + \lambda C + K}{0}{0}{I} .

由于 \det(B - \lambda A) = \det( \lambda^2 M + \lambda C + K )， 我们得出结论：矩阵束 B - \lambda A 是正则的当且仅当 二次矩阵多项式 \lambda^2 M + \lambda C + K 是正则的，并且原QEP (9.1) 的特征值 与矩阵束 B - \lambda A 的特征值一致。

对于特殊情况 (9.2)，我们可以表述 广义特征值问题 Az = \frac{1}{\lambda}Bz，其中

A = \twobytwo{C}{M}{M}{0}, \quadB = \twobytwo{-K}{0}{0}{M}.
 (259)

在这种情况下，两个矩阵都是Hermitian的，但不确定的。 用 (9.15) 进行线性化，在左乘 (9.14) 后，结果是与块对角矩阵 \mathrm{diag}(-I,-M) 相乘。因此，如果 \mathrm{det}(M) \neq 0，那么矩阵束 B - \lambda A 是正则的当且仅当 二次矩阵多项式 \lambda^2 M + \lambda C + K 是正则的。
