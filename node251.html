基本算法

给定一个 n 阶矩阵 A 和初始的 n 行 p 列块向量 P_1 和 Q_1，使得 P^{\ast}_1 Q_1 = I，块Lanczos方法生成一系列 n \times p 的右和左Lanczos块向量 \{ Q_i \} 和 \{ P_i \}。 这些向量是Krylov子空间的双正交基 {\mathcal K}^j(A,Q_1) 和 {\mathcal K}^j(A^{\ast},P_1)。 具体来说， 经过 j 步后，Lanczos过程确定一个块三对角矩阵

T_{[j]} = \left[ \begin{array}{cccc}A_1 & B_2 & & \\C_2......ts & \ddots & B_j \\& & C_j & A_j \\\end{array} \right],

以及右和左Lanczos向量矩阵

Q_{[j]} = \left[\, Q_1, Q_2, \ldots , Q_j \,\right] \quad \m......and} \quadP_{[j]} = \left[\, P_1, P_2, \ldots , P_j \,\right]

满足双正交条件
P^{\ast}_{[j]} Q_{[j]} = I.
 (155)

块Lanczos方法使用三项递归
\displaystyle Q_{j+1} C_{j+1} \textstyle = \displaystyle AQ_j - Q_j A_j - Q_{j-1} B_j, (156)
\displaystyle P_{j+1} B^{\ast}_{j+1} \textstyle = \displaystyle A^{\ast} P_j - P_j A^{\ast}_j - P_{j-1}C^{\ast}_j (157)

对于 j = 1, 2, \ldots，其中 Q_0 = P_0 = 0。 在矩阵表示法中，这些量满足控制关系
\displaystyle A Q_{[j]} \textstyle = \displaystyle Q_{[j]} T_{[j]} + Q_{j+1}C_{j+1}E^{\ast}_j, (158)
\displaystyle A^{\ast} P_{[j]} \textstyle = \displaystyle P_{[j]} T^{\ast}_{[j]} +P_{j+1} B^{\ast}_{j+1}E^{\ast}_j, (159)

其中 E_j 是一个 jp 行 p 列的矩阵，其底部方块是单位矩阵，其他地方为零。

为了使用Lanczos方法 近似 A 的特征值和特征向量，我们解决 jp \times jp 块三对角矩阵 T_{[j]} 的特征值问题。 每个特征三元组 ( \theta^{(j)}_i, z^{(j)}_i, w^{(j)}_i ) 的 T_{[j]}，

T_{[j]} z^{(j)}_i = \theta^{(j)}_i z^{(j)}_i\quad {\rm and......w^{(j)}_i)^{\ast} T_{[j]} = \theta^{(j)}_i (w^{(j)}_i)^{\ast}

确定一个Ritz三元组 ( \theta^{(j)}_i, x^{(j)}_i, y^{(j)}_i )， 其中 x^{(j)}_i = Q_{[j]} z^{(j)}_i 和 y^{(j)}_i = P_{[j]} w^{(j)}_i 。 通常，Ritz三元组首先近似 A 的外部特征三元组。

为了评估近似值， 设 r^{(j)}_i 和 s^{(j)}_i 表示相应的右和左 残差向量：

\displaystyle r^{(j)}_i \textstyle = \displaystyle Ax^{(j)}_i - \theta^{(j)}_i x^{(j)}_i, (160)
\displaystyle (s^{(j)}_i)^{\ast} \textstyle = \displaystyle (y^{(j)}_i)^{\ast} A-\theta^{(j)}_i (y^{(j)}_i)^{\ast}. (161)

分别代入（7.52）和（7.53），出现
\displaystyle r^{(j)}_i \textstyle = \displaystyle Q_{j+1}C_{j+1} (E^{\ast}_j z^{(j)}_i), (162)
\displaystyle (s^{(j)}_i)^{\ast} \textstyle = \displaystyle ((w^{(j)}_i)^{\ast} E_j) B_{j+1} P^{\ast}_{j+1}. (163)

一个Ritz值 \theta^{(j)}_i 被认为已经收敛 如果两个残差范数都足够小。同样， 残差也可以用来确定三元组的后向误差界限， 如同标准非Hermitian Lanczos算法的情况（参见第7.8节）。

我们现在对算法7.14的一些步骤进行评论。

(1)
起始向量 Q_1 和 P_1 最好由用户选择，以体现任何关于 A 所需特征向量的可用知识。在没有此类知识的情况下，选择 Q_1 为一个实正交化的随机 n \times p 矩阵，并设 P_1 = Q_1。

块大小 p 应选择大于或等于任何所需特征值的多重性。如果多重性未知，典型的 p 值为 2、3 和 6。

(2), (3), (20), (21)
用户需要提供子程序来计算任意 n \times p 矩阵 Z 的乘积 AZ 和 A^{\ast}Z。这给用户一个机会，只需一次遍历定义 A 和 A^{\ast} 的数据结构，就可能提高效率。

(8), (9), (10)
首先，计算 R^{\ast} 和 S 的QR分解。如果 R^{\ast} 和 S 都是满秩的，那么 Q_{j+1} 和 P_{j+1} 是 n 行 p 列的矩阵，使得 Q^{\ast}_{j+1} Q_{j+1} = P^{\ast}_{j+1} P_{j+1} = I，并且两者 B^{\ast}_{j+1} 和 C_{j+1} 都是 p 行 p 列的右上三角矩阵。如果 R 或 S 不是满秩的，那么已经揭示了一个不变子空间。在秩亏的情况下继续递归在第7.9.2节中讨论。

(12)
如果 \Sigma_j 是奇异的或接近奇异的， 即，如果有一个零或非常小的奇异值，那么 有一个中断。 必须采取适当的行动继续。参见第7.9.2节以了解可能的处理方法。

(17), (18)
可以使用第7.3节中讨论的方法计算 T_{[j]} 的特征三元组。 Ritz值的收敛性可以通过残差的范数测试，如（7.56）和（7.57）。 有关访问近似精度的更多详细信息，参见第7.8.2节和第7.13节以及[29]。

当 j 变大时，解决 T_{[j]} 的特征问题变得耗时。 一种简单的降低成本的方法是定期进行，例如每五步一次。

(19)
与非块Lanczos方法一样，在有限精度算术的情况下，块Lanczos向量 \{ Q_i \} 和 \{ P_i \} 的双正交性恶化。TSMGS过程可以用来对 Q_{j+1} 和 P_{j+1} 进行就地双正交化 Q_{[j]} =\left[\, Q_1, Q_2, \ldots, Q_j\, \right] 和 P_{[j]} =\left[\, P_1, P_2, \ldots, P_j\, \right]。

PROCEDURE TSMGS
for i=1,2,\ldots,j
Q_{j+1} := Q_{j+1} - Q_i ( P^{\ast}_i Q_{j+1})
P_{j+1} := P_{j+1} - P_i ( Q^{\ast}_i P_{j+1})
end

(25)
这仅在Lanczos向量的双正交性保持良好的情况下推荐。近似特征向量 x^{(j)}_i 和 y^{(j)}_i 对应于收敛的Ritz值 \theta^{(j)}_i 被计算，并且 残差（7.54）和（7.55） 相对于 \Vert x^{(j)}_i\Vert _2 和 \Vert y^{(j)}_i\Vert _2 再次检查。 注意，范数可能大于在步骤（17）中计算的估计值。 这是Lanczos基向量病态的一个副作用。
