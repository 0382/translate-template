特征值与特征向量

多项式 p(\lambda) = {\rm det}(\lambda I-A) 被称为矩阵 A 的 特征多项式 。p(\lambda)=0 的根被称为 A 的 特征值。由于 p(\lambda) 的次数为 n，它有 n 个根，因此 A 有 n 个特征值。实矩阵的特征值可能是实数，也可能成对出现为复数。

满足 Ax = \lambda x 的非零向量 x 是特征值 \lambda 对应的 （右）特征向量。满足 y^*A = \lambda y^* 的非零向量 y 是特征值 \lambda 对应的 左特征向量。

一个 n \times n 的矩阵不一定有 n 个独立的特征向量。最简单的例子是

A (\epsilon) = \bmat{cc} 0 & 1 \\ \epsilon^2 & 0 \emat,
 (3)

其特征值为 \lambda_{\pm} = \pm \epsilon。 当 \epsilon \neq 0 时，有两个右特征向量， x_{\pm} = [ 1, \pm \epsilon ]^T。随着 \epsilon 趋近于 0，这两个特征向量逐渐接近。当 \epsilon = 0 时，两个特征值都等于 0，并且有一个独立的右特征向量平行于 [1,0]^T。n 个独立的特征向量可能不存在（尽管每个不同的特征值至少有一个特征向量），这一事实必然会使非线性特征值问题（NHEP）的理论和算法变得复杂。

由于特征值可能是复数，因此没有固定的方法来排序它们。尽管如此，为了方便，我们通常将它们编号为 \lambda_1 ,\ldots,\lambda_n， 对应的右特征向量为 x_1,\ldots,x_n 和左特征向量为 y_1,\ldots,y_n（如果它们存在）。
