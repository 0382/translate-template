非线性特征值问题
  J. Demmel

本章汇集了几个存在有效算法的非线性特征值问题的实例。由于没有一种单一的方法能解决所有非线性问题，因此每个实例都采用了不同的处理方式。

最简单的非线性特征值问题是二次特征值问题（QEP）

L( \lambda )x = (\lambda^2 M + \lambda B + K)x=0.

这里，\lambda 被称为特征值，x 是（右）特征向量。如果对于所有 \lambda，{\rm det}L( \lambda ) 都不为零，则称为正则的；否则称为奇异的。它可以转换为广义线性特征值问题，如第 3 项在 §2.5.7 和 §2.6.7 中所述。QEP 出现在 §2.2.8 介绍的阻尼弹簧质量系统中：运动方程 M \ddot{x}(t) + B \dot{x}(t) + Kx(t) =0 可以通过代入 x(t) = e^{\lambda t} x 来求解，其中 x 是常向量，\lambda 是待确定的常标量。这会得到如上的 \lambda^2 Mx + \lambda B x + Kx = 0。参见 §9.2。

更高次的多项式特征值问题 (\sum_{i=0}^m \lambda^i A_i)x=0 也可以类似处理。参见 §9.3。

最后，§9.4 考虑了可以表示为在 n 乘 m 正交矩阵 Y 集合上最大化标量函数 F(Y) 的非线性特征值问题。最简单的情况是最大化 F(Y) = Y^* AY，其中 A 是厄米特矩阵且 m=1；答案是 A 的最大特征值。如果 F(Y) = {\rm tr} Y^*AY，即 Y^*AY 的迹或对角线元素之和，那么答案是厄米特矩阵 A 的 m 个最大特征值之和。对于这些问题，存在比这里介绍的基于共轭梯度的优化方案更有效的算法，但其优势在于它可以推广到更多函数 F。我们再给出两个例子。首先，如果 A_1,\ldots,A_m 是 n 乘 n 的实对称矩阵，本应有一组共同的特征向量，但由于噪声干扰不再如此，那么我们寻求一个正交 Y 来最小化所有 Y^*A_iY 的非对角线元素范数之和。其次，在使用局域密度近似的量子力学计算中，希望最大化 {\rm tr}Y^*AY + g(Y)，其中 g(Y) 是一个复杂的非线性项，表示电子间相互作用能量。这里介绍的优化方法可以处理相当一般的函数 g(Y)。
