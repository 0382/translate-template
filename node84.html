谱变换
  R. Lehoucq 和 D. Sorensen

众所周知，上一节中概述的迭代投影方法能迅速提供分离良好的极值特征值的近似值。然而，很多时候，所需的特征值可能并不分离良好，或者位于特征值凸包的内部。在这些情况下，迭代投影方法需要许多步骤才能生成可接受的近似值，如果它们能收敛的话。另一种选择是对A进行谱变换，使得这些分离不良的特征值转化为分离良好的极值特征值。当然，A的特征值应能从变换问题的特征值中轻易恢复，且变换在计算时间和存储方面应高效适用。

分离良好（或非聚集）的特征值的概念可以粗略解释如下。一个分离良好的特征值\lambda_i满足\vert\lambda_i - \lambda_j \vert > \tau \vert\lambda_n - \lambda_1\vert，对于所有j \ne i且\tau \gg\epsilon_M。对于厄米矩阵，我们假设\lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n，而对于非厄米矩阵，\mathfrak{Re}(\lambda_1) \leq \mathfrak{Re}(\lambda_2) \leq \cdots \leq\mathfrak{Re}(\lambda_n)。

通常使用的是移位-反演谱变换（SI）。如果(\lambda,x)是A的一个特征对，Ax = \lambda x，且\sigma \ne \lambda，那么移位-反演特征值问题为

(A - \sigma I)^{-1} x = \nu x, \quad\mathrm{其中} \quad \nu = \frac{1}{\lambda - \sigma }.
 (15)

这种SI对于寻找接近\sigma的特征值非常有效，因为C \equiv (A - \sigma I)^{-1}中幅度最大的近邻特征值\nu_j对应于最接近移位\sigma的近邻特征值\lambda_j。这些变换后幅度最大的特征值正是最容易计算的特征值。一旦找到它们，就可以将它们转换回原始问题的特征值。直接关系为

\lambda_j = \sigma +\frac{1}{\nu_j},

并且与变换问题中的\nu_j相关联的特征向量x_j也是原始问题中对应于\lambda_j的（广义）特征向量。其他谱变换技术包括广义凯莱变换；详见[324]。

SI在迭代步骤（即子空间的维度）方面极其有效，应尽可能使用。特别是在寻找内部特征值，或者所需特征值的幅度明显小于幅度最大的特征值，或者所需特征值聚集的情况下。

谱变换的一个潜在缺点是必须求解涉及A-\sigma I的线性系统。这可以通过矩阵分解或迭代方法来完成。用户应在可行的情况下使用稀疏直接方法对适当的矩阵进行分解。如果使用迭代方法求解线性系统，解的精度必须与特征值求解器的收敛容差相匹配。一种启发式方法是，相对于所需的特征值计算精度，迭代线性系统求解需要稍严格的容差。详见[283,291,414]。

也可以以近似方式进行变换，并使用不再具有Krylov结构的子空间。这是几种算法背后的思想，包括Jacobi-Davidson方法。后者及其他方法将在本书的其余部分详细讨论。
