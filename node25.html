特征分解

定义 \Lambda = {\rm diag}(\lambda_1 ,\ldots, \lambda_n ) 和 X = [x_1,\ldots,x_n]。X 被称为 A 的特征向量矩阵。 由于 x_i 是正交单位向量，我们可以看到 X^*X=I；即， X 是一个酉（正交）矩阵。 这 n 个等式 A x_i = \lambda_i x_i 对于 i=1,\ldots,n 也可以写成 AX = X \Lambda 或 A = X \Lambda X^*。这种分解

A = X \Lambda X^*

被称为 A 的特征分解。换句话说， A 与对角矩阵 \Lambda 相似，相似变换为 X。

如果我们取 X 的 k 列（例如 \hat{X}= X(:,[2,3,5]) = 第 2、3 和 5 列），那么这些列张成 A 的不变子空间。 如果我们取相应的子矩阵 \hat{\Lambda}= {\rm diag}(\lambda_2 , \lambda_3 , \lambda_5 ) 的 \Lambda，那么我们可以写出相应的 部分特征分解 为 A \hat{X}= \hat{X}\hat{\Lambda} 或 \hat{X}^* A \hat{X}= \hat{\Lambda}。如果 \hat{X} 中的列被 k 个不同的向量替换，这些向量张成相同的不变子空间，那么我们得到 一个不同的部分特征分解 A \check{X}= \check{X}\check{A}， 其中 \check{A} 是 一个 k \times k 矩阵，其特征值与 \hat{\Lambda} 相同，尽管 \check{A} 可能不是对角矩阵。
